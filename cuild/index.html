What is "cuild"?

cuild (short for CBuild) is a tiny build system for the C/C++ programming language. It's focused in ease of use and simplicity, with the level of comprehension of a simple programming language.

Can I see an example of this technology?

Down below you have a demonstration of how simple Cuild is.

compiler(gcc)
project(my_cuild_project)

add_file(main.c)
link_lib(glfw)
link_lib(GL)
add_cflag(std=c99)
add_cflag(Wall)

You can also build libraries.

compiler(gcc)
project(qg)

add_obj(qg)
add_obj(qglfw)
add_cflag(Wall)
make_static(libqg)
